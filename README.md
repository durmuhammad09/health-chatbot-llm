# General Health Query Chatbot (LLM Project)

## 📌 Task Objective
The objective of this project is to build a simple AI chatbot that can answer general health-related questions using a Large Language Model (LLM). The chatbot provides friendly and clear responses using prompt engineering and includes basic safety filtering.

---

## 🛠 Tools & Technologies Used
- Python
- Google Colab
- Hugging Face Transformers
- Pretrained Language Model (FLAN-T5 / QA model)

---

## 📊 Dataset Used
No dataset was manually used.

The chatbot uses a **pretrained Large Language Model**, so it already contains learned medical/general knowledge.

---

## 🤖 Models Applied
- Question Answering Model from Hugging Face Transformers
- Pretrained LLM (FLAN-T5 / QA pipeline)

---

## ⚙️ Features
- Accepts user health questions
- Provides simple and friendly answers
- Uses prompt engineering to improve responses
- Safety filter to block harmful questions
- Beginner-friendly chatbot interface

---

## 🔍 Example Queries
- What causes a sore throat?
- Is paracetamol safe for children?
- What are symptoms of fever?

---

## 🚫 Safety Handling
The chatbot blocks harmful queries such as:
- self-harm
- overdose
- dangerous medical instructions

It advises users to consult a doctor when needed.

---

## 📈 Key Results & Findings
- The chatbot successfully responds to general health questions
- Prompt engineering improves response clarity
- Safety filter helps prevent harmful outputs
- Pretrained LLM can be used without training dataset

---

## 🧠 Conclusion
This project demonstrates how AI and LLMs can be used to build a simple health assistant chatbot using Python and Google Colab.

It is suitable for beginners and internship-level AI/ML learning.
